# nodejs-api
Proyecto para Kazam

$ npm install

$ brew install mongodb

$ brew cask install robo-3t

$ sudo mkdir -p /data/db

$ sudo chown -R `id -un` /data/db

Abrir una terminal y correr:

$ mongod

Para sair: ctrl-c


Configuramos mongo para usar API

$ mongo 

Dentro del interprete, escribimos:

> use api

> exit
